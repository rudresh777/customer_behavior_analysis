# 🛒 Retail Customer Behavior Analysis Project

## 📌 Overview

This project demonstrates a complete end-to-end **Data Analyst workflow** using real-world retail customer behavior data. It covers data cleaning, SQL analysis, dashboard creation, and reporting—similar to tasks performed in a professional business environment.

The goal is to extract meaningful insights about customer purchasing patterns and support data-driven decision-making.

---

## 📂 Dataset

* Dataset Name: *Customer Shopping Behavior Dataset*
* Contains information such as:

  * Customer demographics (age, gender)
  * Purchase details (items, category, amount)
  * Payment methods
  * Discounts applied
  * Purchase frequency

---

## 🛠️ Tools & Technologies

* **Python (Pandas, NumPy)** → Data Cleaning & Preprocessing
* **SQL (PostgreSQL / MySQL / SQL Server)** → Data Analysis
* **Power BI** → Interactive Dashboard
* **Gamma** → Presentation (PPT)
* **Excel** → Initial data review

---

## ⚙️ Steps / Process

### 1. Data Cleaning (Python)

* Removed missing and duplicate values
* Standardized column names
* Converted data types
* Created new features (e.g., age groups)

### 2. Data Loading (SQL)

* Created database and tables
* Imported cleaned dataset into SQL Server
* Verified schema and data integrity

### 3. SQL Analysis

Performed business-focused queries such as:

* Top-selling products by category
* Customer segmentation by age group
* Discount impact on sales
* Most preferred payment methods

### 4. Dashboard Creation (Power BI)

* Built interactive visuals:

  * Sales by category
  * Customer demographics
  * Discount analysis
  * Purchase trends
* Added slicers for dynamic filtering

### 5. Reporting

* Summarized insights into a structured report
* Highlighted key business findings

### 6. Presentation (Gamma)

* Created a professional PPT
* Included charts, insights, and recommendations

---

## 📊 Dashboard

Key features of the Power BI dashboard:

* Interactive filters (gender, category, age group)
* KPI cards (total sales, total customers)
* Visualizations:

  * Bar charts for product performance
  * Pie charts for category distribution
  * Line charts for trends

---

## 📈 Results / Insights

* Certain product categories generate the highest revenue
* Discounts significantly increase purchase rates
* Specific age groups show higher spending behavior
* Digital payment methods are most preferred

## ▶️ How to Run

### 1. Clone Repository

```bash
git clone https://github.com/rudresh777/customer_behavior_analysis.git
```

### 2. Run Python Script

```bash
python data_cleaning.py
```

### 3. Load Data into SQL

* Open SQL Server 
* Create database
* Import cleaned CSV file

### 4. Run SQL Queries

* Execute queries from `/sql/analysis.sql`

### 5. Open Power BI Dashboard

* Open `.pbix` file in Power BI Desktop

### 6. View Presentation

* Open Gamma PPT file
